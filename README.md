# Simple TCP Client-Server System

A Python-based TCP socket communication system that simulates a temperature sensor sending data to a central server.

## How to Run
1. **Start the Server:** Open a terminal and run `python server.py`.
2. **Start the Client:** Open a second terminal and run `python client.py`.
3. The client will generate a random temperature and send it to the server every 5 seconds.

## Test Results
- **Localhost:** Tested using 127.0.0.1; communication successful.
- **Network Test:** Tested using IP 10.207.50.1 over local WiFi; data received correctly.
